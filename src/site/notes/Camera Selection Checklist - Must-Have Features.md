# Camera Selection Checklist - Must-Have Features for Intermediate Photographers

*Based on analysis of 7 YouTube reviews and expert recommendations*

## 🎯 Priority Features (Must-Haves)

### ✅ Autofocus System
- [ ] **Phase Detection Autofocus** (not contrast-based)
- [ ] **Eye Detection AF** for portraits
- [ ] **Animal Eye AF** if shooting wildlife/pets
- [ ] **Fast AF acquisition** (<0.5 seconds)
- [ ] **Good low-light AF performance** (works in dim conditions)
- [ ] **Subject tracking** capability

### ✅ Image Stabilization
- [ ] **In-Body Image Stabilization (IBIS)** - 5+ stops preferred
- [ ] **Electronic stabilization** as minimum backup
- [ ] **Works with any lens** (advantage of IBIS over lens-based)

### ✅ Sensor & Image Quality
- [ ] **APS-C minimum** (avoid smaller sensors)
- [ ] **24-33MP resolution** (sweet spot for most needs)
- [ ] **Good high ISO performance** (usable to ISO 6400+)
- [ ] **14-bit RAW support** for editing flexibility

### ✅ Build Quality & Ergonomics
- [ ] **Weather sealing** (dust/moisture resistant)
- [ ] **Dual card slots** (critical for important work)
- [ ] **Good grip and button layout**
- [ ] **Electronic Viewfinder (EVF)** for bright conditions
- [ ] **Articulating LCD screen** (flip-out preferred)

### ✅ Connectivity & Power
- [ ] **USB-C charging** (modern standard)
- [ ] **Good battery life** (400+ shots per charge)
- [ ] **Wi-Fi connectivity** for image transfer

## 🔍 System Considerations (Ecosystem)

### Lens Availability
- [ ] **Large lens selection** (Sony E-mount leads)
- [ ] **Affordable third-party options** (Sigma, Tamron)
- [ ] **Future lens roadmap** from manufacturer
- [ ] **Used lens market availability**

### Long-term Investment
- [ ] **Active system development** (not discontinued)
- [ ] **Regular firmware updates**
- [ ] **Strong user community** and support
- [ ] **Upgrade path available** within same system

## 📊 Budget Allocation Priorities

### Recommended Spending Distribution:
1. **50% on lenses** (most important for image quality)
2. **35% on camera body**
3. **15% on accessories** (cards, batteries, tripod)

### Lens Priority Order:
- [ ] **Primary lens first** (50mm equivalent or 24-70mm zoom)
- [ ] **Avoid kit lenses** - invest in better glass
- [ ] **Wide aperture options** (f/1.4-f/2.8 for shallow DOF)

## 🚫 Features to Avoid/Deprioritize

### Don't Prioritize:
- [ ] ~~Megapixel count above 40MP~~ (unnecessary for most)
- [ ] ~~Latest/newest model~~ (previous gen often better value)
- [ ] ~~Kit lens bundles~~ (usually poor quality)
- [ ] ~~Brand loyalty~~ (choose best system for needs)

### Red Flags:
- [ ] ~~Contrast-only autofocus~~
- [ ] ~~No image stabilization~~
- [ ] ~~Single card slot for professional use~~
- [ ] ~~Proprietary battery/charging~~
- [ ] ~~Limited lens ecosystem~~

## 📋 Pre-Purchase Research Checklist

### Before Buying:
- [ ] **Watch comparison videos** on YouTube
- [ ] **Check used market prices** (B&H, MPB, KEH)
- [ ] **Read user reviews** on forums/Reddit
- [ ] **Test in store** if possible
- [ ] **Consider rental** for expensive purchases
- [ ] **Check firmware version** and update availability

### Questions to Ask Yourself:
- [ ] What type of photography will I do most?
- [ ] How important is portability vs. image quality?
- [ ] Do I need professional features or enthusiast level?
- [ ] What's my total system budget (body + lenses + accessories)?
- [ ] Am I buying for current needs or future growth?

## 💡 Final Tips

### Smart Shopping:
- [ ] **Buy used from reputable dealers** (30-40% savings)
- [ ] **Focus on lens quality** over camera features
- [ ] **Consider previous generation** cameras (better value)
- [ ] **Factor in total system cost** when choosing brand

### Common Beginner Mistakes to Avoid:
- [ ] Obsessing over specs instead of using camera
- [ ] Buying too much gear too quickly
- [ ] Ignoring the importance of good lighting
- [ ] Not investing enough in post-processing skills
- [ ] Choosing camera based on brand reputation alone
